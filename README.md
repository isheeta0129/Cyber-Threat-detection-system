# Cyber-Threat-detection-system
A web-based cyber threat detection system that monitors suspicious activities, detects potential cyber attacks, and alerts administrators using Python and machine learning.
# Cyber Threat Detection System

## Project Description
This project monitors suspicious activities and detects cyber threats.

## Features
- Detect suspicious login attempts
- Monitor IP activity
- Alert administrators

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Machine Learning

## How to Run the Project
1. Install Python
2. Run backend server
3. Open website dashboard
